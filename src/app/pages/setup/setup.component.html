DigiGoat Activation

DigiGoat should open automatically, but if not click here to begin the setup process.

DigiGoat will open up to a setup window, make sure that it says "Configure" if you already started to make your website, and if not it should say "Get Started". That's it! You're good to go now!

If you're still encountering issues, please make sure that DigiGoat is installed. If it is currently open, try closing it and then reloading this page. If it is not currently open, try opening it and then reloading this page. If you're still encountering issues, reach out to support@digigoat.app for assistance
